Introduction

Here are some visual representations of all of the current Pokemon! The bar chart charts the number of pokemon in any given generation, while the pie chart shows the overall split in the primary type of all pokemon.
Using the menus, and clicking on the charts, you can see the splits of the pokemon and the spread they have. It's interesting to see how types are split through each generation and even more interesting when you see breakdowns of the legendary pokemon, with the more popular types used in each generation.
Play around with the data and see how each generation of Pokemon was made up! Which is the most popular type in each generation? How many fire type legendaries are there? and across which generations?
